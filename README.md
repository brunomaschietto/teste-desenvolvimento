## 🕹 **Como rodar esse projeto?**

```
# Clone esse repositório
$ git clone link-repositório

# Acesse a pasta do projeto no seu terminal
$ cd teste-dev

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run dev
```
